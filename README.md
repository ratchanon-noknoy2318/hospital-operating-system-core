## Overview

Official hospital web system used in a real healthcare environment to support staff workflows with intuitive interfaces for clinical operations under high workload and time-sensitive conditions.

---

## UX Problem

Healthcare staff often face challenges with existing systems:

- Interfaces are complex and difficult to navigate  
- Workflows require excessive steps and repeated input  
- Poor layout design increases cognitive load  
- Critical information is not clearly presented  

These issues slow down operations and increase the risk of user errors.

---

## UX Approach

The system was designed with a workflow-first approach:

- Simplified user flows based on real hospital processes  
- Reduced steps in common tasks such as patient handling  
- Structured layouts for faster scanning and interaction  
- Clear grouping of related information  
- Designed for high-frequency usage by staff  

---

## Core UI Concepts

- Form-driven workflows optimized for speed  
- Clear visual hierarchy for critical information  
- Minimal and consistent interface design  
- Responsive layout for different usage environments  
- Task-focused screens aligned with real operations  

---

## User Flow

Design → Code Implementation → Responsive Iteration → Deployment  → Staff Usage
